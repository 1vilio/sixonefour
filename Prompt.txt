Контекст : Пишем десктопное приложение для сайта soundcloud.com. Будем использовать один из самых популярных опен сорс репозиториев soundcloud-rpc.
В нём уже реализовано много нужного функционала,вот что он имеет : Десктопный клиент SoundCloud с поддержкой Discord Rich Presence, тёмной темы, Last.fm и AdBlock,а так же шорткаты для настройки всех функций.
Он был написан с использованием : 
xhayper/discord-rpc - Discord Rich Presence integration 
Electron - Framework for building cross-platform desktop applications
electron-builder - Tool for packaging and distributing Electron applications
ghostery/adblocker-electron - Ad blocking functionality 
electron-store - Data persistence
electron-updater - Auto-update functionality
