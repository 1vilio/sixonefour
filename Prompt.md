Привет, Gemini 3!

Мы продолжаем разработку десктопного клиента для SoundCloud. Этот проект создан с использованием Electron и TypeScript и нацелен на расширение стандартных возможностей SoundCloud за счет добавления уникальных функций.

**Основная цель:** Создать многофункциональный, настраиваемый и удобный десктопный клиент для SoundCloud.

**Технологический стек:**
- **Фреймворк:** Electron
- **Язык:** TypeScript
- **Сборка и зависимости:** Node.js, pnpm (используя workspaces)
- **UI:** Чистый HTML, CSS и JavaScript.

**Архитектура и структура проекта:**
Проект имеет модульную структуру, где каждая ключевая функция вынесена в отдельный "сервис". Это позволяет гибко управлять функционалом.
- `src/main.ts`: Главный процесс Electron. Здесь происходит инициализация приложения, создание окон (основного и виджета) и подключение всех сервисов.
- `src/services/*.ts`: Ядро приложения. Каждый файл в этой директории отвечает за отдельную функцию:
    - `themeService.ts`: Управляет кастомными темами оформления (инъекция CSS).
    - `shortcutService.ts`: Отвечает за глобальные горячие клавиши.
    - `presenceService.ts`: Интеграция с Discord Rich Presence.
    - `downloadService.ts`: Реализует скачивание треков.
    - `listeningStatsService.ts`: Собирает статистику прослушиваний.
    - `WidgetManager.ts`: Управляет десктопным виджетом.
    - `translationService.ts`: Отвечает за интернационализацию (i18n).
    - `settingsManager.ts`: Управляет сохранением и загрузкой настроек пользователя.
- `src/header/`, `src/widget/`, `src/statistics/`: Директории, содержащие код (HTML/CSS/JS) для соответствующих частей интерфейса.
- `themes/`: Папка для пользовательских CSS-тем.
- `assets/`: Иконки, фоновые видео и другие ресурсы.

**Ключевые реализованные функции:**
1.  **Система кастомных тем:** Пользователи могут создавать и применять свои `.css` файлы для полного изменения внешнего вида приложения.
2.  **Десктопный виджет:** Полупрозрачный виджет с эффектом "стекла" (glassmorphism), который отображает информацию о текущем треке, обложку и элементы управления. Поддерживает видео-фоны.
3.  **Глобальные горячие клавиши:** Возможность управлять воспроизведением (пауза/плей, следующий/предыдущий трек) из любой точки операционной системы.
4.  **Скачивание треков:** Функция для загрузки музыки напрямую из SoundCloud.
5.  **Discord Rich Presence:** Отображение текущего прослушиваемого трека в статусе Discord.
6.  **Статистика прослушиваний:** Сбор данных о том, какие треки и как долго слушает пользователь.
7.  **Настройки при запуске:** Опции для автозапуска приложения вместе с системой и запуска в трее.
8.  **Интернационализация (i18n):** Поддержка нескольких языков интерфейса.

1.  **Улучшить метод сбора статистики прослушиваний:** Текущая система работает, но требует доработки для большей точности и сбора расширенных данных.
2.  **Локализация и выбор языка:** Добавить новые языки, обновить существующие переводы и дать пользователю возможность выбирать язык интерфейса в настройках.

5.  **Обучающий тур:** Реализовать краткое обучение для новых пользователей, которое знакомит с основными функциями при первом запуске.
